# 城市建設點擊遊戲 (City Builder Clicker) - 遊戲設計企劃

## 1. 遊戲背景與故事 (Story & Theme)
**故事大綱**：
你是一名充滿野心的拓荒者，帶著簡單的工具來到了一片充滿豐富自然資源的無人荒野。你的目標是從無到有，將這片荒野發展成一個繁華的現代大都會。
- **起步**：你必須親手撿拾枯枝、敲打岩石，解決溫飽。
- **發展**：隨著資源累積，你開始搭建伐木場、農場與礦坑，並吸引流浪者加入你的營地，建立社會體系。
- **工業與繁榮**：當你建立了「銀行」與「發電廠」，你將從一個勞動者轉變為「市長」與「資本家」，透過金錢僱用大批工人，甚至研發「自動化機器」來取代人力，建立起你的高科技城市帝國。

---

## 2. 核心資源系統 (Resource System)
遊戲包含多種互相牽制的資源，加入了生存壓力與高科技發展，讓策略更具深度：

1. 🪵 **木頭 (Wood)**：最基礎的建材。
2. 🪨 **石頭 (Stone)**：堅固的進階建材。
3. ⚙️ **金屬 (Metal)**：工業化的關鍵材料。
4. 🍞 **食物 (Food)**：**生存核心資源**。
   - **來源**：手動採集、建造「農田 (Farm)」、「牧場 (Ranch)」。
   - **用途**：養活工人。**每個工人每秒會消耗一定數量的食物**。
   - **⚠️ 飢荒機制 (Starvation)**：如果食物歸零，工人將會開始餓死（人口下降），導致所有依賴人力的產能瞬間崩潰！擴張前必須先確保糧食產量。
5. 💰 **金錢 (Money)**：經濟命脈，用來僱用工人、購買科技。由「銀行」或市場交易產生。
6. ⚡ **電力 (Power)**：後期核心資源。
   - **來源**：建造「發電廠 (Power Plant)」。
   - **用途**：維持高級建築（如研究所、自動化機器）的運作。電力不足會導致設備停工。
7. 🔬 **科技點數 (Science)**：由「研究所 (Research Lab)」產生，用於解鎖科技樹與自動化設備。
8. 👷 **工人 (Workers)**：城市的勞動力，需要金錢僱用，消耗食物生存。
9. 🤖 **自動化機器 (Automated Machines)**：後期的勞動力替代品，**不消耗食物，但消耗電力**，產能極高。

## 3. 遊戲階段與核心循環 (Game Phases & Loop)

### 第一階段：荒野求生與溫飽（純手動點擊）
- **點擊機制**：畫面中央有「森林」、「岩石」與「灌木叢（找食物）」。
- 玩家必須手動點擊收集木頭、石頭與食物。
- **目標**：存活下來，並建造出第一座「農田」來確保穩定的食物來源，避免自己餓死。

### 第二階段：聚落成型（初級自動化）
- 玩家建造了「農田」與「牧場」確保食物後，開始建造伐木場與採石場。
- 建造「木屋 (Housing)」增加人口上限，並開始**僱用少量工人**。
- 玩家必須在「資源生產」與「糧食消耗」之間抓平衡，不能無腦狂招工人。

### 第三階段：資本崛起與城市化（銀行與公寓）
- 玩家建造了**「銀行 (Bank)」**，開始產生金錢。
- 為了容納爆炸性成長的勞動力，玩家解鎖並建造**「公寓 (Apartment)」**，大幅提升工人上限。
- 玩家介面解鎖「人力調度面板」，將幾百名工人動態分配到伐木、挖礦、種田、放牧。這時期考驗玩家對「糧食網」的掌控能力。

### 第四階段：工業革命與自動化（電力與研究所）
- **發電廠**的落成標誌著工業時代的到來。城市開始有電力供應。
- 建造**「研究所 (Research Lab)」**，消耗電力與金錢，產生科技點數。
- **終極目標**：研發出**「自動化機器」**。將勞動力從脆弱的人類（會餓死）轉移到機器人（吃電）上，實現產能的幾何級數爆發！

## 4. 建築物列表 (Buildings)

| 建築名稱 | 功能描述 | 消耗/需求 |
| :--- | :--- | :--- |
| **農田 (Farm)** | 自動生產農作食物 (Food/sec)。基礎糧食來源。 | 無 |
| **牧場 (Ranch)** | 自動生產大量肉類食物。產量高，但可能需要消耗少量水或木頭維持。 | 無 |
| **伐木場 / 採石場 / 礦坑** | 自動生產木頭 / 石頭 / 金屬。 | 需要工人操作才能最大化產量 |
| **木屋 (Housing)** | 提供少量工人人口上限。 | 木頭 |
| **公寓 (Apartment)** | 提供**大量**工人人口上限，節省建築空間。 | 石頭、金屬 |
| **銀行 (Bank)** | 自動生產金錢 (Money/sec)。 | 石頭、金屬 |
| **發電廠 (Power Plant)** | 產生電力 (Power 總容量)。有火力、風力、核能等升級。 | 金屬、(燃料) |
| **研究所 (Research Lab)** | 每秒產生「科技點數」，解鎖科技樹。 | ⚡ 消耗電力 |
| **機器人工廠 (Robotics Factory)** | 生產「自動化機器」。 | 金錢、金屬、⚡消耗電力 |

## 5. 升級與科技樹系統 (Tech Tree & Upgrades)

除了基礎的「工具升級」與「勞工法案」，研究所將解鎖龐大的科技樹：
1. **基因改造農作 (GMO Crops)**：大幅降低工人餓死的風險，農田產能翻倍。
2. **高壓電網 (High-Voltage Grid)**：提升發電廠效率，支援更多高科技建築。
3. **AI 運算中心 (AI Core)**：終極科技，解鎖最高階自動化機器，機器的產能加成基於你累積的科技點數總量。

---

## 6. 特殊機制：飢荒與停電災難 (Disasters)

- **飢荒 (Starvation)**：遊戲的懲罰機制。如果過度擴張導致食物產量 (Food CPS) < 工人消耗量，且食物庫存歸零，工人將會每秒以一定百分比死亡。死亡的工人會導致資源鏈崩潰，玩家必須手動介入挽救（例如：緊急解僱工人、手動點擊找食物）。
- **停電 (Blackout)**：當用電需求超過發電量，所有依賴電力的設施（研究所、自動化機器）會全面停擺。玩家必須手動關閉部分工廠，或緊急升級發電廠來恢復供電。
