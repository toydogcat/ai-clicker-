# 城市建設點擊遊戲 (City Builder Clicker) - 遊戲設計企劃

## 1. 遊戲背景與故事 (Story & Theme)
**故事大綱**：
你是一名充滿野心的拓荒者，帶著簡單的工具來到了一片充滿豐富自然資源的無人荒野。你的目標是從無到有，將這片荒野發展成一個繁華的現代大都會。
- **起步**：你必須親手撿拾枯枝、敲打岩石。
- **發展**：隨著資源累積，你開始搭建伐木場、礦坑，並吸引流浪者加入你的營地。
- **繁榮**：當你建立了「銀行」與金融體系，你將從一個勞動者轉變為「市長」與「資本家」，透過金錢僱用大批工人，建立起你的城市帝國。

---

## 2. 核心資源系統 (Resource System)
遊戲包含多種互相牽制的資源，讓玩家有策略升級的空間：

1. 🪵 **木頭 (Wood)**：最基礎的建材。
   - **來源**：手動點擊「森林」、建造「伐木場」。
   - **用途**：建造基礎設施、房屋。
2. 🪨 **石頭 (Stone)**：堅固的進階建材。
   - **來源**：手動點擊「碎石地」、建造「採石場」。
   - **用途**：建造更大型的建築（如銀行、大型工廠）。
3. ⚙️ **金屬 (Metal)**：工業化的關鍵材料。
   - **來源**：建造「礦坑」後才能開採。
   - **用途**：升級工具（提升手動點擊效率）、建造高級自動化設備。
4. 💰 **金錢 (Money)**：經濟命脈與後期核心資源。
   - **來源**：建造「銀行 (Bank)」後解鎖，透過市場交易或稅收隨時間自動產生。
   - **用途**：**僱用工人**、購買強力的社會政策與高級科技。
5. 👷 **工人 (Workers)**：城市的勞動力。
   - **來源**：花費「金錢」進行僱用，且需要有足夠的「房屋」提供居住上限。
   - **用途**：可以自由分配到伐木、採石或採礦工作上，大幅提供該資源的每秒產量 (CPS)。

---

## 3. 遊戲階段與核心循環 (Game Phases & Loop)

### 第一階段：荒野求生（純手動點擊）
- **點擊機制**：畫面中央有兩個主要點擊區塊：「森林邊緣」與「岩石堆」。
- 玩家必須手動瘋狂點擊，每次點擊獲得 1 木頭 或 1 石頭。
- **目標**：收集足夠的資源來建造第一座自動化建築（例如：需要 50 木頭解鎖「簡陋伐木場」）。

### 第二階段：聚落成型（初級自動化）
- 玩家擁有了伐木場與採石場，遊戲開始有微量的自動收益 (CPS)。
- 玩家解鎖「金屬」資源，必須存資源建造「礦坑」。
- 透過金屬可以購買「鐵斧」、「鐵鎬」等升級，讓手動點擊的收益從 1 變成 5 甚至 10。

### 第三階段：資本崛起（銀行與僱傭系統）
- 這是遊戲的**第一個大高潮**。玩家花費大量石頭與金屬建造了宏偉的**「銀行 (Bank)」**。
- 銀行開始印製鈔票（產生金錢）。
- 玩家介面解鎖「人力市場」。玩家可以花費金錢僱用工人。
- 玩家可以隨時調整工人的分配：例如將 10 個工人派去伐木，5 個派去挖礦，讓資源產出可以根據當下需求進行**動態微調**。

## 4. 建築物列表 (Buildings)

| 建築名稱 | 消耗資源 | 功能描述 |
| :--- | :--- | :--- |
| **伐木場 (Lumber Camp)** | 木頭 | 自動生產木頭 (Wood/sec)。 |
| **採石場 (Quarry)** | 木頭、石頭 | 自動生產石頭 (Stone/sec)。 |
| **礦坑 (Mine)** | 木頭、石頭 | 自動生產金屬 (Metal/sec)。 |
| **帳篷/木屋 (Housing)** | 木頭 | 提供人口上限，讓你可以僱用更多工人。 |
| **銀行 (Bank)** | 石頭、金屬 | **核心建築**。自動生產金錢 (Money/sec)。升級銀行可以提高利息與產能。 |
| **市場 (Market)** | 木頭、金屬 | 允許玩家以較差的匯率將木頭/石頭/金屬直接賣成金錢。 |

---

## 5. 升級系統 (Upgrades)

升級分為幾大類：
1. **工具升級 (Tool Upgrades)**：消耗金屬。將你的點擊工具從「石器」升級為「鐵器」、「鋼器」，大幅提升每次手動點擊的資源量。
2. **效率手冊 (Efficiency Books)**：消耗金錢。提升所有自動化建築 10% ~ 50% 的產出。
3. **勞工法案 (Labor Laws)**：消耗大量金錢。提升每個工人的工作效率（例如：原本一個工人 1 木頭/秒，升級後變成 2 木頭/秒）。

## 6. 特殊機制構想 (未來可以加入的細節)

- **隨機事件 (如同黃金餅乾)**：
  - **「遊商路過」**：畫面上出現一台馬車，點擊它可以獲得一筆巨大的金錢，或是以極低價格買到大量金屬。
  - **「資源豐收」**：短暫時間內，伐木場或採石場的產量翻倍。
- **時代演進 (Prestige/飛升機制)**：
  - 當城市發展到極致（現代化），玩家可以選擇「建立新市鎮」（重置遊戲）。
  - 重置後玩家會獲得「市政經驗值 / 傳奇藍圖」，讓下一輪的遊戲有永久性的加成（例如：建築成本降低、基礎工人產能翻倍）。
